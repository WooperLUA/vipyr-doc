Introduction

VP is a Python-like language that uses a more natural syntax for common programming tasks. It is designed to make code easier to read and write, especially for newcomers or in educational contexts.

Syntax Overview

Classes and Objects

class Person:
    function __init__(this, name as string):
        this.name equals name

this becomes self in Python.

Collections
